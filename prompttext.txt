You need to program a Survey in the plain2Form language. The questionaire you get has the following structure:

The first row of the table with Fragenummer shows the Name of the questions. If the name starts with a number or the letter Q you need to add F before the number
The next row kurzbeschreibung can be ignored
The row Filter shows the showif condition to be programmed. In case the Filter says All no showif is needed.
For Filter purposes the first value in a single question is always 1 and in a multiple always A1. 
The Filtervalue is only dependent of the position and not on the number in code. This means the Filter for Keine Angabe is not 98.
If there is a Screenout for a specific answer put [SCREENOUT] behind that answer. If a Screenout is a more complex condition use jumpif:CONDITION {SCREENOUT}
If under weitere Anmerkung there is rotieren the answer possibilities should be randomised by putting [random:a] behind them but not the Answer Keine Angabe. This should always tbe last. Answers with inputs should also not be randomised
Auswertung should be ignored. 
The Row after is the Text of the Questions. 
After the question come the possible answers. If there is a _______ an [input] should be programmed and the lines not programmed. Multiple questions can include multiple inputs.
In single choice questions input should always have [exclusive]
If there is only an input field and just one answer like Keine Angabe make the input big with: [input:m]
The is only makro per row. This means if there is a input and showif they are in one bracket.
If an input field is only numbers use [input;numeric;decimal_places:0], decimal_places shows all the allowed numbers of decimals.
To  limit the allowed number use: input;numeric: >= 14 && <= 99 or input;numeric: >=0
Keine Angabe in single questions should have [code:98] behind it. 
Multiple questions should have [exlusive;deselect] behind it if the answer is exclusive.
To get the max number of allowed questions use: quantity: > 0 && <= 3 allows between 1 and 3 answers.
If a questions only shown if a certain number of answers were selected use this format: showif: F01A1 + F01A2 ... > 1
If there is text with no answers use [interviewer]
If the first row after the question has a rating skala and the following row is an answer it is a rating question and should be programmed as such.
If all the answer possibilites are ratings it is a single questions and should contain numbers.
If there is a | in Keine Angabe | Weiß nicht replace it with a /
Always translate the error message in the language of the questions.
