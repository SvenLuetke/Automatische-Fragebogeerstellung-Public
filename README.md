# Automatische-Fragebogeerstellung

Erstellt automatisch einen plain2form Fragebogenformat basierend auf einen Word Fragebogen. Jede Frage muss in einer Tabelle sein. Genauerer Aufbau der Fragen kann in dem Prompt an die KI beschrieben werden. 
Es gibt ein Beispieltext für den Prompt der angepasst werden kann. 
Verwendet wird hier die GEMINI API und dementsprechend wir ein GEMINI API KEY benötigt.
